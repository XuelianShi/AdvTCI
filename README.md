## üìñ Introduction

This repository contains the official implementation of the paper: **"Adversarial Triangular Chromatic Illumination: A Stealthy Physical Attack on Object Detectors"**.

**AdvTCI** is a novel black-box physical attack method that utilizes **Particle Swarm Optimization (PSO)** to generate adversarial perturbations by optimizing a parameterized colored triangular illumination. Unlike traditional patch-based attacks, AdvTCI is **non-contact**, **stealthy**, and capable of achieving high attack success rates against advanced object detectors (e.g., YOLOv5, Faster R-CNN) in both digital and physical environments.
